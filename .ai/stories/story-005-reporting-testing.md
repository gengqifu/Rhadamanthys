# 用户故事：报告生成与测试验证
- 状态：Complete
- 故事点：3 SP

## 背景
需要生成符合要求的报告，并提供测试与示例验证工具可用性。

## 需求
- 报告：默认 Excel，字段含规则ID、风险等级、文件/行、证据、建议、是否需人工复核；风险配色红/黄/绿；默认排序风险降序、规则ID升序；额外覆盖统计 Sheet。
- 可选输出 JSON/CSV，通过参数开启，路径沿用 `--out` 前缀。
- 证据格式：文件路径+行号+命中片段，中文命中理由与建议。
- 提供示例项目与示例报告，验证安装和运行。
- 关键用例测试：权限文案缺失、ATS 全关、无 ATT+跟踪 SDK、第三方登录缺少苹果登录、外链支付/第三方支付 SDK、私有 API 命中、明文 HTTP、后台模式滥用、规则库缺失/加载错误。

## 验收标准
- 生成的 Excel 满足字段、排序、配色与覆盖统计要求。
- JSON/CSV 可选输出正确生成。
- 示例项目可跑通，关键用例被检测并出现在报告中。
